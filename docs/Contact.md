## 📝 Submission Guidelines

To help our engineers resolve your issue as quickly as possible, please include the following in your report:

!!! info "Required Information"
    * **Environment:** Operating System (e.g., Ubuntu 22.04, Windows 11).
    * **Versions:** Software version or Git commit hash.
    * **Logs:** Paste relevant error logs using code blocks ( \` \` \` ).
    * **Reproduction:** A clear list of steps to recreate the bug.
